MODULO 07 - PARTE 06 - LETRA C

Criar uma página que contenha um alarme. Para isso, utilizar o HTML select para seleção dos minutos e segundos (máx 59 min), criar um botão que inicia a contagem regressiva que ao ser clicado vira um botão de desarmar. O contador deve mostrar o tempo a cada segundo, quando faltar menos de 5% do tempo inicial, mudar a interface e informar que o tempo está acabando.

Ao término da contagem, deve-se disparar um alarme informando que o tempo acabou. Se o usuário clicar no botão de desarmar alarme, a contagem deve ser parada e, caso o alarme esteja tocando, o som deve parar, transformando o botão para que possa armar o alarme novamente.
