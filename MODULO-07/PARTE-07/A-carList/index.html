Crie uma página web que possua um elemento HTML select que possa selecionar uma lista de carros esportivos, podendo identificar a mudança de carro selecionado e alterar a imagem associada, exibir as informações referentes a este carro (dados do carro como nome, fabricante, velocidade máxima, 0-100km e outras informações que julgar pertinente)
